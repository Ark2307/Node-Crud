->UNDERSTANDING the DEPENDENCIES

* Express.js
It is a framework based on Node.js. 
It has important features required for web applications.
The implemented code of some important HTTP module is present already in Express.js.

* Morgan
It is a Node. js and Express middleware to log HTTP requests and errors.
Middleware is a function that has access to the request and response lifecycle methods.
The next() method to continue logic in your Express server.

* Nodemon
Nodemon is a tool that helps develop node. js based applications by automatically restarting the node application when file changes in the directory are detected.

* EJS 
EJS is a simple templating language which is used to generate HTML markup with plain JavaScript.
It also helps to embed JavaScript to HTML pages.
EJS is a tool for generating web pages that can include dynamic data.
It can share templated pieces with other web pages.

* Body-Parser
Express body-parser is an npm library used to process data sent through an HTTP request body.
It exposes four express middlewares for parsing text, JSON, url-encoded and raw data set through an HTTP request body. 
These middlewares are functions that process incoming requests before they reach the target controller.

* Mongoose
Mongoose is a Object Data Modeling (ODM) library for MongoDB distributed as an npm package.

* Axios
Axios is a Javascript library used to make HTTP requests from node. js 
It enables client-side protection against XSRF.
It also has the ability to cancel requests.
